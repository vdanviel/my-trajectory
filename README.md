# 💼 Professional Journey Project

This repository contains a personal project developed to present my professional journey in a clear, visual, and structured way.

## 🚀 Purpose

This project was created to:

* Serve as a **professional portfolio**
* Demonstrate my **technical and problem-solving skills**
* Provide a **clear narrative of my career progression**
* Help recruiters and other developers understand my background quickly

## 🛠️ Technologies Used

* Next.js
* React
* Tailwind CSS

## 📈 Features

* Timeline-style visualization of my journey
* Clean and responsive UI
* Customizable theme (colors, branding, etc.)
